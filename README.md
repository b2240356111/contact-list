#  Simple Contact Manager

A Python-based command-line application designed to manage personal contacts efficiently. This tool allows you to perform basic CRUD (Create, Read, Update, Delete) operations through an interactive menu.

---

###  Key Features

* **Add New Contact:** Save contacts with their name, an 11-digit phone number, email, and country.
* **View Contacts:** Display all saved contacts in a clean, tabular format including their index, name, and details.
* **Search & Edit:** Find contacts by name (or part of the name) and update specific fields like phone number or email.
* **Secure Deletion:** Delete contacts with a confirmation prompt to prevent accidental data loss.
* **Input Validation:** Built-in checks for empty fields, proper phone number length, and menu selection.

---

###  How to Use

1. **Run the Script:**
   Open your terminal and run the following command:
   python contact_manager.py
   
2. **Navigation**
   Use numbers 1-5 to navigate the main menu.
   Type 'cancel' at any input prompt to safely return to the main menu without saving changes.
